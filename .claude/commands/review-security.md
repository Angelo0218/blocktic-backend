description: 安全審查 - 檢查 OWASP Top 10 和個資保護
---

掃描整個 src/ 目錄，檢查以下安全問題：

1. **注入攻擊**: 是否有 raw SQL 未參數化、未驗證的使用者輸入
2. **認證/授權**: JWT 驗證是否完善、是否有未保護的端點
3. **個資洩漏**: 是否有 personIdHash 以外的個資被記錄到日誌或回傳給前端
4. **敏感資訊**: .env 是否在 .gitignore 中、是否有硬編碼的密鑰
5. **CORS/Helmet**: 是否有正確的 HTTP 安全標頭設定
6. **Rate Limiting**: 關鍵端點是否有限流
7. **SBT 合約安全**: 合約中 transfer 覆寫是否正確阻擋非 owner 操作

以表格呈現發現的問題和嚴重程度。
